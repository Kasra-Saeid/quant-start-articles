Advanced Algorithmic Trading and QSTrader - Second Update

Advanced Algorithmic Trading and QSTrader - Second Update

This is a quick update post to let readers know that the pre-order release of Advanced Algorithmic Trading has had a new update, adding over 50 pages of material. This brings the current release up to 250 pages. To access the new content, customers simply need to follow the download link received in the original purchase email. If the download email has been misplaced then please email support@quantstart.com and the updated version will be sent out.

In addition the open source backtesting framework, QSTrader, has had significant development in the past few months. A lot of work has gone into improving the backtesting execution speed, by optimising the usage of Decimal datatypes for position calculations. The API for running a backtest has improved drastically, with major changes to the configuration capability.

I would personally like to thank both @ryankennedyio and @femtotrader for making excellent suggestions, giving so much of their free time to the project and sharing their changes with the wider quant community. While QSTrader remains in "alpha" state, it is shaping up to become an extremely useful software tool for quantitative trading.

If you have any questions regarding Advanced Algorithmic Trading, QSTrader or any other aspect of quantitative finance then please email support@quantstart.com and the team will get back to you.
